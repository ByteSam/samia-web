# examples/

Pon aquí fragmentos de código reales que quieras que Claude Code copie como patrón al construir algo nuevo (no hace falta que sean archivos completos del proyecto — este repo ya tiene el código real en `src/`; esta carpeta es para snippets externos, ejemplos de otra librería, o patrones que quieras fijar explícitamente).

Si no hay nada aquí, Claude Code igual puede referenciar patrones directos de `src/components/` y `src/app/` — esta carpeta es un extra, no un requisito.
