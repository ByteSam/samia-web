# Documentación oficial de SamIA

Esta carpeta es la fuente de verdad para las decisiones vigentes del negocio y del sitio. Antes de crear una página, cambiar una oferta o ajustar el copy, se debe revisar esta documentación.

Los documentos anteriores ubicados fuera de este repositorio se conservan como material de referencia. No son la fuente oficial cuando contradicen estos archivos.

## Orden de lectura

1. [Decisiones vigentes](./00-decisiones-vigentes.md)
2. [Oferta y servicios](./01-oferta-y-servicios.md)
3. [Web y conversión](./02-web-y-conversion.md)

## Regla de actualización

Una decisión que afecte la oferta, el mensaje de la web, el embudo, los precios o la arquitectura debe actualizarse aquí en el mismo cambio. Las cifras no confirmadas se marcan como `por definir`; nunca se inventan para completar una página comercial.
